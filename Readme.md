# Ziploy Github Action

Automate deployment of your WordPress projects from Github to any hosting with Ziploy.

## Includes

-   Ubuntu 24.04 Minimal
-   PHP 8.4.16
-   Composer 2.8.5
-   Node v23.7.0
-   NPM 10.9.2
-   yarn 1.22.22
-   git 2.43
-   curl 8.5.0

## Variables

-   `$ZIPLOY_HOST`
    Full address of your website, eg: `https://my.wordpress-host.com`

-   `$ZIPLOY_ID`
    ID of your ziploy project you want to deploy, eg: `123`

-   `$ZIPLOY_SECRET`
    Secret key from your ziploy project, eg: `myr$nd0M5tr1nG`
